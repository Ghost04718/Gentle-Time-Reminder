# Gentle Time Reminder

Gentle Time Reminder is a friendly Chrome extension that helps you stay aware of time passing. See [here](https://chromewebstore.google.com/detail/gentle-time-reminder/ohoebhfifcpnhkhoihjlnmadbmamdinj).

## Features

- Customizable reminder intervals
- Gentle notifications that do not disrupt your workflow
- Easy to enable or disable reminders
- Simple and intuitive user interface

## Installation

1. Download the extension from the Chrome Web Store.
2. Click on the "Add to Chrome" button.
3. Confirm the installation by clicking "Add extension" in the pop-up window.

## Usage

1. Click on the Gentle Time Reminder icon in the Chrome toolbar.
2. Set your desired reminder interval.
3. Enable the reminders by toggling the switch.
4. You will receive gentle notifications at the set intervals.

## Notes

- The maximum snooze time is 180 minutes.
- The notification interval can be set between 1 and 180 minutes.
- The notification duration is fixed at 30 seconds.
- The extension supports up to two default snooze options: 10 minutes and 30 minutes.
- The extension can be paused and resumed at any time.
- Sound notifications can be enabled or disabled.
